# Isotropic 3D cardiac cine imaging at 0.55T using stack-of-spiral sampling and four-dimensional iterative motion correction (4D iMoCO) 

This project is a Gadgetron extension that reconstructions stack-of-spiral cine acquisitions with a 4D iMoCo.

## Publication 

This open-source code is described in :
- Rajiv Ramasawmy, Ahsan Javed, Daniel A. Herzka, Prakash Kumar, Krishna S. Nayak, Robert J. Lederman, Adrienne E. Campbell-Washburn
- Isotropic 3D cardiac cine imaging at 0.55T using stack-of-spiral sampling and four-dimensional iterative motion correction (4D iMoCO)
. In preparation
